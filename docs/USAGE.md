# Управление Eyes in the Dark

Мод создаёт клиентский хоррор-эффект, которым управляет сервер. Призрак не является мобом, не получает урон и не наносит его игрокам.

## Быстрая проверка

```mcfunction
/eyesinthedark spawn
/eyesinthedark force
/eyesinthedark blink
/eyesinthedark clear
```

`spawn` появляется за исполнителем команды. `force` запускает девятисекундную сцену взгляда. `blink` заставляет глаза мигнуть по краю зрения и убежать. `clear` сразу убирает эффект.

## Команды для сервера

Все команды требуют уровень прав `2`.

```mcfunction
/eyesinthedark spawn [игроки] [natural|idle|stare|flee|vanish|blink]
/eyesinthedark force [игроки]
/eyesinthedark blink [игроки]
/eyesinthedark clear [игроки]
/eyesinthedark chance [0.20..1.0]
/eyesinthedark enabled [true|false]
/eyesinthedark interval [200..24000]
/eyesinthedark cooldown [200..72000]
/eyesinthedark status
```

`chance` меняет шанс автоматической встречи, но не позволяет опустить его ниже 20%. `enabled` включает и выключает автоматические появления. `interval` задаёт частоту проверок в тиках, а `cooldown` - задержку до следующей автоматической встречи.

## Автоматический режим

По умолчанию проверка запускается раз в 2400 тиков. Для встречи игрок должен быть в выживании, ниже Y=48, без неба над головой и при уровне освещения не выше 5. Призрак появляется в 8-14 блоках, не подходит ближе 7.5 блоков во время сцены взгляда и отступает, если игрок оказывается ближе 6 блоков.

Настройки находятся в `config/eyesinthedark-common.toml` после первого запуска сервера.
